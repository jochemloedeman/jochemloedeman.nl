# jochemloedeman.nl

## Technologies
This is the codebase for a static photography portfolio website. It is built with HTML, Tailwind CSS and plain JavaScript.

## Page setup
The website contains three pages:
- A homepage (index.html). On this page, the user can choose to visit either of the two photography pages (Urban or Nature). The options are presented as two representative images side-by-side, filling the main view. When the user hovers over these images, the image becomes more transparent and a text with the name of the category appears over the image. Clicking on either image will navigate the user to the respective photography page.
- A urban photography page (urban.html). On this page, the user can view a collection of urban photography images. The images are displayed in a grid layout, and clicking on an image will open it in a larger view.
- A nature photography page (nature.html). On this page, the user can view a collection of nature photography images. The images are displayed in a grid layout, and clicking on an image will open it in a larger view.
